# @electripy/cli

NPM wrapper package for ElectriPy CLI tools.

## Status

**This is a placeholder package.** The actual ElectriPy tools are written in Python.

## Installation

To use ElectriPy, install the Python package:

```bash
pip install electripy
```

## Usage

After installing the Python package, you can use the CLI:

```bash
electripy doctor    # Check installation health
electripy version   # Show version
electripy --help    # Show help
```
