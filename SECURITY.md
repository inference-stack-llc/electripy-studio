# Security Policy

## Supported versions

| Version | Supported |
| --- | --- |
| 0.x (latest) | ✅ |
| < latest 0.x | ❌ |

## Reporting a vulnerability

**Please do not open a public issue for security vulnerabilities.**

Instead, email **[security@inferencestack.dev](mailto:security@inferencestack.dev)** with:

1. A description of the vulnerability
2. Steps to reproduce
3. Affected versions
4. Any suggested fix (optional)

We will acknowledge receipt within **48 hours** and aim to release a fix within **7 days** for critical issues.

## Scope

This policy covers the `electripy` Python package published from this repository. Third-party dependencies are out of scope, but we appreciate reports of vulnerable transitive dependencies.
